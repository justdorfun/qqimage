## 产品功能
－ 特效：16种精选特效，LOMO、移轴任你选；
－ 相框：精美相框不限量，在线相框源源不断；
－ 裁剪：各种比例，极其流畅，剪裁照片也能不一样；
－ 旋转：旋转，翻转都可以，还有自由旋转等你来发现；
－ 调色：亮度、对比度、饱和度任意调节，调节过度不再是个问题。
－ 即拍即用，拍好照片就能美化和分享。
－ 微信，QQ空间，腾讯微博，新浪微博，人人网，电子邮件，Facebook，Flickr，多个平台，快乐齐分享。
